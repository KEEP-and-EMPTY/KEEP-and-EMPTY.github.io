## 3.git中的操作

// ssh密钥

ssh-keygen -t rsa -C “2837468248@qq.com“（此处应填为你建立GitHub的邮箱）

（此步骤后就要去GitHub建立公开密钥）

// 连结

ssh -T git@github.com

// 安装hexo

npm install -g hexo

// hexo初始化

hexo init

// 安装deployer

npm install hexo-deployer-git –save

hexo g

hexo d

（初次建立可能要求你输入你的GitHub的账号密码)
