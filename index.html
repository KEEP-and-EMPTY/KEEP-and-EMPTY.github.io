​ 学的时候，也不要被一个教程给缠死了，要多方浏览

​ 不放弃的话自己也是肯定也可以做成的。加油！

2.初次搭建基于GitHub的基本过程

1.环境搭载

1.安装nodejs可以去官网上下载

​ 下载的具体过程一般就是一直下一步

2.安装git当然也是去官网下载

​ 下载的具体过程一般就是一直下一步

2.GitHub上的操作

1.建立一个仓库

2.建立公开密钥

3.git中的操作

// ssh密钥

ssh-keygen -t rsa -C “2837468248@qq.com“（此处应填为你建立GitHub的邮箱）

（此步骤后就要去GitHub建立公开密钥）

// 连结

ssh -T git@github.com

// 安装hexo

npm install -g hexo

// hexo初始化

hexo init

// 安装deployer

npm install hexo-deployer-git –save

hexo g

hexo d

（初次建立可能要求你输入你的GitHub的账号密码)

​ (我失败一次后重来，不知道为什么就不要了)
